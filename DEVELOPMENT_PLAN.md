# План разработки — Приложение "Список элементов"

## Обзор проекта

Веб-приложение с двумя панелями для управления списком из 1 000 000 элементов.
Левая панель — невыбранные элементы, правая — выбранные с возможностью сортировки через Drag&Drop.

**Технологии:** Express.js (бэкенд), Vue 3 (фронтенд), TypeScript.

---

## Этап 1: Инициализация проекта

- [ ] Создать монорепо-структуру (`/server`, `/client`)
- [ ] Настроить TypeScript для обоих проектов
- [ ] Настроить package.json с общими скриптами запуска

## Этап 2: Бэкенд (Express.js + TypeScript)

### 2.1 Хранилище данных (in-memory)
- [ ] Инициализация массива из 1 000 000 элементов (ID 1–1 000 000)
- [ ] Структура: `Set<number>` для выбранных элементов, `Map` для порядка сортировки
- [ ] Хранение добавленных пользователем элементов

### 2.2 API-эндпоинты
- [ ] `GET /api/items` — получение невыбранных элементов (пагинация: offset + limit=20, фильтр по ID)
- [ ] `GET /api/selected` — получение выбранных элементов (пагинация + фильтр, с учётом сортировки)
- [ ] `POST /api/items/select` — выбор элемента (перемещение в правое окно)
- [ ] `POST /api/items/deselect` — отмена выбора (перемещение в левое окно)
- [ ] `POST /api/items/add` — добавление нового элемента с произвольным ID
- [ ] `PUT /api/selected/reorder` — изменение порядка выбранных элементов (Drag&Drop)

### 2.3 Очередь запросов с дедупликацией
- [ ] Реализовать серверную очередь для обработки запросов
- [ ] Дедупликация: одно и то же значение не добавляется повторно
- [ ] Батчинг: добавление элементов — раз в 10 секунд
- [ ] Батчинг: получение и изменение данных — раз в 1 секунду

## Этап 3: Фронтенд (Vue 3 + TypeScript)

### 3.1 Общая архитектура
- [ ] Двухпанельный layout (левое/правое окно)
- [ ] Composable для API-запросов с очередью и дедупликацией на клиенте

### 3.2 Левое окно (невыбранные элементы)
- [ ] Компонент списка с infinite scroll (загрузка по 20 элементов)
- [ ] Поле фильтрации по ID
- [ ] Кнопка выбора элемента (перемещение в правое окно)
- [ ] Форма добавления нового элемента с произвольным ID

### 3.3 Правое окно (выбранные элементы)

- [ ] Поле фильтрации по ID
- [ ] Drag&Drop сортировка (работает и для отфильтрованного списка)
- [ ] Кнопка отмены выбора (перемещение в левое окно)
- [ ] Сохранение состояния выбора и сортировки между обновлениями страницы

### 3.4 Клиентская очередь запросов
- [ ] Очередь с дедупликацией на клиенте
- [ ] Батчинг добавления — раз в 10 секунд
- [ ] Батчинг получения/изменения — раз в 1 секунду
